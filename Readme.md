# 🚀 SaaS Landing Page - Practice Project

A modern, responsive landing page for a SaaS (Software as a Service) platform. This project was built to practice clean UI/UX design, responsive layouts, and organized folder structures.

## 🔗 Live Demo 🔗
https://69bc371f2ac58f065bac7ea9--charming-sprite-817790.netlify.app/

## ✨ Features
* **Modern UI:** Clean aesthetic with a focus on readability.
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop.
* **Optimized Assets:** Compressed images and minified code structures.
* **Interactive Elements:** Smooth scrolls and hover effects using JavaScript.

## 🛠️ Built With
* **HTML5:** Semantic structure for better SEO.
* **CSS3:** Flexbox/Grid for layout and custom animations.
* **JavaScript:** Form handling and mobile menu functionality.
